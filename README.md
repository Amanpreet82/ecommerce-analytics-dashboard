# ecommerce-analytics-dashboard
This project presents an interactive E-Commerce Dashboard built using Power BI to analyze sales performance, revenue trends, and customer behavior.
The dashboard helps businesses make data-driven decisions by providing clear insights into key metrics like revenue, average order value, and cancellation rates.
 Dashboard Features
🔹 Revenue Analysis
Revenue breakdown by Category (Laptop, Mobile, Headphones, etc.)
Top-performing categories: Laptop, Mobile
🔹 Time-Based Trends
Revenue analysis by Year, Quarter, and Day
Helps identify seasonal trends and sales patterns
🔹 Regional Analysis
State-wise revenue distribution
Key contributing states include:
Maharashtra
Gujarat
Rajasthan
🔹 Product Performance
Top products by revenue:
HP Pavilion
Sony Headphones
MacBook Air
iPhone 13
Objectives
Analyze overall sales and revenue performance
Identify top-performing products and categories
Track revenue trends over time
Understand regional (state-wise) performance
Monitor business KPIs
